# legacy/ — آرشیوِ UIِ قدیمی (tkinter)

این فایل‌ها **UIِ نسلِ اولِ پروژه** (مبتنی بر tkinter) هستند و دیگر استفاده نمی‌شوند.
UIِ فعلی، نسخه‌ی حرفه‌ایِ **PySide6** در پوشه‌ی `ui/` است که از طریق `app.py` (entrypointِ بسته‌بندی) یا
`python -m ui.app_qt` اجرا می‌شود.

| فایل | توضیح |
|------|-------|
| `gui.py` | UIِ tkinter اصلی. حاویِ الگوی `_ensure_admin()` (ShellExecuteW runas) بود که الان در `core/admin.py` به‌صورت خالص و قابل‌تست بازنویسی شده. |
| `gui_old2.py` | نسخه‌ی آزمایشیِ قدیمی‌ترِ tkinter. |

**چرا حذف کامل نشد؟** برای مرجعِ تاریخی و نگاشتِ منطق (مثلاً جزئیاتِ اتصال به xray/vwarp)
نگه داشته شده. در صورت نیاز می‌توان آن‌ها را کاملاً حذف کرد — هیچ بخشی از کدِ فعال به آن‌ها وابسته نیست.
